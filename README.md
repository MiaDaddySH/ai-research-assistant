# AI Research Assistant

A small AI project that researches a topic by searching the web, reading multiple sources, and generating a concise report.

## MVP Features
- Enter a research question
- Search the web for relevant articles
- Fetch article content
- Summarize findings with Azure OpenAI
- Return a final research summary with sources

## Tech Stack
- Frontend: Flutter
- Backend: FastAPI
- LLM: Azure OpenAI
- Search: Tavily
- Parsing: httpx + BeautifulSoup

## Project Structure
- frontend/: Flutter app
- backend/: FastAPI app
